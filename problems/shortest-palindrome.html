Shortest Palindrome

LeetCode Problem N/A
Problem: You are given a string s. You can convert s to a palindrome by adding characters in front of it.

Solution 1 â€“ KMP Algorithm

Use KMP algorithm to find the longest palindromic prefix.

public String shortestPalindrome(String s) { String temp = s + "#" + new StringBuilder(s).reverse().toString(); int[] table = getTable(temp); return new StringBuilder(s.substring(table[table.length - 1])).reverse().toString() + s; } private int[] getTable(String s) { int[] table = new int[s.length()]; int j = 0; for (int i = 1; i < s.length(); i++) { while (j > 0 && s.charAt(i) != s.charAt(j)) { j = table[j - 1]; } if (s.charAt(i) == s.charAt(j)) { j++; } table[i] = j; } return table; }
Time Complexity: O(n)
Space Complexity: O(n)
