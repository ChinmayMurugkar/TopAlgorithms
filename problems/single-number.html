Single Number

LeetCode Problem 136
Problem: Given a non-empty array of integers nums, every element appears twice except for one. Find that single one.

Constraints:
  • 1 ≤ nums.length ≤ 3 * 10⁴
  • -3 * 10⁴ ≤ nums[i] ≤ 3 * 10⁴
  • Each element in the array appears twice except for one element which appears only once.

Example:
Input: nums = [2,2,1]
Output: 1

Solution 1 – Hash Set

Use a hash set to track seen numbers. If a number is seen twice, remove it from the set.

public int singleNumber(int[] nums) { Set set = new HashSet<>(); for (int num : nums) { if (set.contains(num)) { set.remove(num); } else { set.add(num); } } return set.iterator().next(); }
Time Complexity: O(n)
Space Complexity: O(n)

Solution 2 – XOR (Optimal)

Use XOR operation. XOR of a number with itself is 0, and XOR with 0 is the number itself.

public int singleNumber(int[] nums) { int result = 0; for (int num : nums) { result ^= num; } return result; }
Time Complexity: O(n)
Space Complexity: O(1)
XOR Properties:
• a ⊕ a = 0 (number XORed with itself gives 0)
• a ⊕ 0 = a (number XORed with 0 gives the number)
• a ⊕ b ⊕ a = b (XOR is associative and commutative)
XOR Walkthrough:

Input: [4, 1, 2, 1, 2]

result = 0
result = 0 ⊕ 4 = 4 (binary: 100)
result = 4 ⊕ 1 = 5 (binary: 101)
result = 5 ⊕ 2 = 7 (binary: 111)
result = 7 ⊕ 1 = 6 (binary: 110)
result = 6 ⊕ 2 = 4 (binary: 100)

Final result: 4

Solution 3 – Math Approach

Use mathematical formula: 2 * (sum of unique elements) - sum of all elements = single number.

public int singleNumber(int[] nums) { Set uniqueNums = new HashSet<>(); int sum = 0; for (int num : nums) { uniqueNums.add(num); sum += num; } int uniqueSum = 0; for (int num : uniqueNums) { uniqueSum += num; } return 2 * uniqueSum - sum; }
Time Complexity: O(n)
Space Complexity: O(n)

Variations

Single Number II (LeetCode 137)

Every element appears three times except for one. Find that single one.

public int singleNumber(int[] nums) { int ones = 0, twos = 0; for (int num : nums) { ones = (ones ^ num) & ~twos; twos = (twos ^ num) & ~ones; } return ones; }

Single Number III (LeetCode 260)

Every element appears twice except for two elements. Find those two single numbers.

public int[] singleNumber(int[] nums) { int xorResult = 0; // XOR all numbers for (int num : nums) { xorResult ^= num; } // Find the rightmost set bit int rightmostSetBit = xorResult & -xorResult; int x = 0, y = 0; // Divide numbers into two groups for (int num : nums) { if ((num & rightmostSetBit) != 0) { x ^= num; } else { y ^= num; } } return new int[]{x, y}; }

public boolean isPowerOfTwo(int n) { return n > 0 && (n & (n - 1)) == 0; }

public boolean isPowerOfFour(int n) { return n > 0 && (n & (n - 1)) == 0 && (n & 0xAAAAAAAA) == 0; }
