Palindrome Linked List

LeetCode Problem 234
Problem: Given the head of a singly linked list, return true if it is a palindrome or false otherwise.

Example:
Input: head = [1,2,2,1]
Output: true
Explanation: The list reads the same forwards and backwards.

Solution 1 – Reverse Second Half (Optimal)

Find the middle, reverse the second half, and compare with the first half.

public boolean isPalindrome(ListNode head) { if (head == null || head.next == null) { return true; } // Find the middle of the list ListNode slow = head; ListNode fast = head; while (fast.next != null && fast.next.next != null) { slow = slow.next; fast = fast.next.next; } // Reverse the second half ListNode secondHalf = reverseList(slow.next); ListNode firstHalf = head; // Compare the two halves while (secondHalf != null) { if (firstHalf.val != secondHalf.val) { return false; } firstHalf = firstHalf.next; secondHalf = secondHalf.next; } return true; } private ListNode reverseList(ListNode head) { ListNode prev = null; ListNode current = head; while (current != null) { ListNode nextTemp = current.next; current.next = prev; prev = current; current = nextTemp; } return prev; }
Time Complexity: O(n)
Space Complexity: O(1)

Solution 2 – Stack Approach

Use a stack to store the first half and compare with the second half.

public boolean isPalindrome(ListNode head) { if (head == null || head.next == null) { return true; } // Find the middle ListNode slow = head; ListNode fast = head; while (fast.next != null && fast.next.next != null) { slow = slow.next; fast = fast.next.next; } // Push first half onto stack Stack stack = new Stack<>(); ListNode current = head; while (current != slow.next) { stack.push(current.val); current = current.next; } // Compare with second half if (fast.next == null) { stack.pop(); // Remove middle element for odd length } while (current != null) { if (stack.pop() != current.val) { return false; } current = current.next; } return true; }
Time Complexity: O(n)
Space Complexity: O(n)
Reverse Second Half Visualization:

Step 2: Reverse second half

Second half: 1 ← 2 (reversed)

Step 3: Compare
