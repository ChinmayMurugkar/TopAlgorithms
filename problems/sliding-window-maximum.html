Sliding Window Maximum

LeetCode Problem N/A
Problem: You are given an array of integers nums, there is a sliding window of size k which is moving from the very left of the array to the very right.

Solution 1 – Deque

Use deque to maintain maximum elements in sliding window.

public int[] maxSlidingWindow(int[] nums, int k) { if (nums == null || nums.length == 0) return new int[0]; int n = nums.length; int[] result = new int[n - k + 1]; Deque deque = new LinkedList<>(); for (int i = 0; i < n; i++) { // Remove elements outside window while (!deque.isEmpty() && deque.peekFirst() < i - k + 1) { deque.pollFirst(); } // Remove smaller elements while (!deque.isEmpty() && nums[deque.peekLast()] < nums[i]) { deque.pollLast(); } deque.offerLast(i); if (i >= k - 1) { result[i - k + 1] = nums[deque.peekFirst()]; } } return result; }
Time Complexity: O(n)
Space Complexity: O(k)
