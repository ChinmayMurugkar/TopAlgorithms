Remove Nth Node From End of List

LeetCode Problem 19
Problem: Given the head of a linked list, remove the nth node from the end of the list and return its head.

Example:
Input: head = [1,2,3,4,5], n = 2
Output: [1,2,3,5]
Explanation: Remove the 2nd node from the end (node with value 4).

Solution 1 – Two Pointers (Optimal)

Use two pointers with a gap of n nodes to find the node to remove.

public ListNode removeNthFromEnd(ListNode head, int n) { ListNode dummy = new ListNode(0); dummy.next = head; ListNode first = dummy; ListNode second = dummy; // Move first pointer n+1 steps ahead for (int i = 0; i <= n; i++) { first = first.next; } // Move both pointers until first reaches end while (first != null) { first = first.next; second = second.next; } // Remove the nth node from end second.next = second.next.next; return dummy.next; }
Time Complexity: O(n)
Space Complexity: O(1)

Solution 2 – Two Pass Approach

First pass to count nodes, second pass to remove the target node.

public ListNode removeNthFromEnd(ListNode head, int n) { // Count total nodes int count = 0; ListNode current = head; while (current != null) { count++; current = current.next; } // Calculate position from start int position = count - n; // Handle case where we need to remove head if (position == 0) { return head.next; } // Find the node before the one to remove current = head; for (int i = 0; i < position - 1; i++) { current = current.next; } // Remove the node current.next = current.next.next; return head; }
Time Complexity: O(n)
Space Complexity: O(1)
Two Pointers Visualization:

List: 1 → 2 → 3 → 4 → 5, n = 2

Step 1: Create dummy node
dummy → 1 → 2 → 3 → 4 → 5

Step 2: Move first pointer n+1 steps (3 steps)
first = 4, second = dummy

Step 3: Move both pointers until first reaches end
first = null, second = 3

Step 4: Remove node after second (node 4)
Result: 1 → 2 → 3 → 5
