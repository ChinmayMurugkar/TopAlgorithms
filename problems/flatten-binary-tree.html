Flatten Binary Tree to Linked List

LeetCode Problem 114
Problem: Given the root of a binary tree, flatten the tree into a "linked list" in-place. The "linked list" should use the same TreeNode class where the right child pointer points to the next node in the list and the left child pointer is always null.

Example:
Input: root = [1,2,5,3,4,null,6]
Output: [1,null,2,null,3,null,4,null,5,null,6]

Solution 1 – Recursive (Reverse Preorder)

Flatten the right subtree, then the left subtree, and rewire the pointers.

public void flatten(TreeNode root) { flattenHelper(root, null); } private TreeNode flattenHelper(TreeNode node, TreeNode prev) { if (node == null) { return prev; } prev = flattenHelper(node.right, prev); prev = flattenHelper(node.left, prev); node.right = prev; node.left = null; prev = node; return prev; }
Time Complexity: O(n)
Space Complexity: O(h) where h is the height of the tree

Solution 2 – Iterative with Stack

Use a stack to simulate preorder traversal and rewire pointers.

public void flatten(TreeNode root) { if (root == null) { return; } Stack stack = new Stack<>(); stack.push(root); while (!stack.isEmpty()) { TreeNode current = stack.pop(); if (current.right != null) { stack.push(current.right); } if (current.left != null) { stack.push(current.left); } if (!stack.isEmpty()) { current.right = stack.peek(); } current.left = null; } }
Time Complexity: O(n)
Space Complexity: O(h)
Flattening Process:

Before:
1
/ \
2 5
/ \ \
3 4 6

After:
1 → 2 → 3 → 4 → 5 → 6
