Isomorphic Strings

LeetCode Problem 205
Problem: Given two strings s and t, determine if they are isomorphic.

Definition: Two strings s and t are isomorphic if the characters in s can be replaced to get t.

Rules:
  • All occurrences of a character must be replaced with another character
  • No two characters may map to the same character
  • A character may map to itself

Example:
Input: s = "egg", t = "add"
Output: true
Explanation: e → a, g → d

Solution 1 – Two Hash Maps

Use two hash maps to ensure bidirectional mapping is consistent.

public boolean isIsomorphic(String s, String t) { if (s.length() != t.length()) { return false; } Map sToT = new HashMap<>(); Map tToS = new HashMap<>(); for (int i = 0; i < s.length(); i++) { char sChar = s.charAt(i); char tChar = t.charAt(i); // Check if sChar already maps to a different character if (sToT.containsKey(sChar) && sToT.get(sChar) != tChar) { return false; } // Check if tChar already maps to a different character if (tToS.containsKey(tChar) && tToS.get(tChar) != sChar) { return false; } sToT.put(sChar, tChar); tToS.put(tChar, sChar); } return true; }
Time Complexity: O(n)
Space Complexity: O(n)

Solution 2 – Array-based Mapping

Use arrays instead of hash maps for better performance since we're dealing with characters.

public boolean isIsomorphic(String s, String t) { if (s.length() != t.length()) { return false; } int[] sToT = new int[256]; int[] tToS = new int[256]; Arrays.fill(sToT, -1); Arrays.fill(tToS, -1); for (int i = 0; i < s.length(); i++) { char sChar = s.charAt(i); char tChar = t.charAt(i); // Check if mapping is consistent if (sToT[sChar] == -1 && tToS[tChar] == -1) { sToT[sChar] = tChar; tToS[tChar] = sChar; } else if (sToT[sChar] != tChar || tToS[tChar] != sChar) { return false; } } return true; }
Time Complexity: O(n)
Space Complexity: O(1) since arrays are fixed size

Solution 3 – Single Pass with Last Occurrence

Track the last occurrence of each character to ensure consistent mapping.

public boolean isIsomorphic(String s, String t) { if (s.length() != t.length()) { return false; } int[] sLastIndex = new int[256]; int[] tLastIndex = new int[256]; Arrays.fill(sLastIndex, -1); Arrays.fill(tLastIndex, -1); for (int i = 0; i < s.length(); i++) { char sChar = s.charAt(i); char tChar = t.charAt(i); // Check if characters have the same last occurrence pattern if (sLastIndex[sChar] != tLastIndex[tChar]) { return false; } sLastIndex[sChar] = i; tLastIndex[tChar] = i; } return true; }
Time Complexity: O(n)
Space Complexity: O(1)
More Examples:

Solution 4 – Character Transformation

Transform both strings to a normalized form and compare.

public boolean isIsomorphic(String s, String t) { return transformString(s).equals(transformString(t)); } private String transformString(String str) { Map charToIndex = new HashMap<>(); StringBuilder result = new StringBuilder(); for (int i = 0; i < str.length(); i++) { char c = str.charAt(i); if (!charToIndex.containsKey(c)) { charToIndex.put(c, charToIndex.size()); } result.append(charToIndex.get(c)); } return result.toString(); }
Time Complexity: O(n)
Space Complexity: O(n)

Edge Cases and Validation

Robust implementation handling edge cases:

public boolean isIsomorphic(String s, String t) { // Handle null inputs if (s == null || t == null) { return s == t; } // Handle different lengths if (s.length() != t.length()) { return false; } // Handle empty strings if (s.length() == 0) { return true; } int[] sToT = new int[256]; int[] tToS = new int[256]; Arrays.fill(sToT, -1); Arrays.fill(tToS, -1); for (int i = 0; i < s.length(); i++) { char sChar = s.charAt(i); char tChar = t.charAt(i); // Check for mapping conflicts if (sToT[sChar] == -1) { if (tToS[tChar] != -1) { return false; // tChar already mapped to different char } sToT[sChar] = tChar; tToS[tChar] = sChar; } else if (sToT[sChar] != tChar) { return false; // sChar maps to different char } } return true; }
