Longest Common Subsequence

LeetCode Problem 1143
Problem: Given two strings text1 and text2, return the length of their longest common subsequence.

Definition: A subsequence of a string is a new string generated from the original string with some characters (can be none) deleted without changing the relative order of the remaining characters.

Example:
Input: text1 = "abcde", text2 = "ace"
Output: 3
Explanation: The longest common subsequence is "ace" and its length is 3.

Solution 1 – Dynamic Programming (Bottom-up)

Use a 2D DP table where dp[i][j] represents the length of LCS of text1[0...i-1] and text2[0...j-1].

public int longestCommonSubsequence(String text1, String text2) { int m = text1.length(); int n = text2.length(); int[][] dp = new int[m + 1][n + 1]; for (int i = 1; i <= m; i++) { for (int j = 1; j <= n; j++) { if (text1.charAt(i - 1) == text2.charAt(j - 1)) { dp[i][j] = dp[i - 1][j - 1] + 1; } else { dp[i][j] = Math.max(dp[i - 1][j], dp[i][j - 1]); } } } return dp[m][n]; }
Time Complexity: O(m*n)
Space Complexity: O(m*n)
DP Table Example:
text1 = "abcde", text2 = "ace"
a c e
0 0 0 0
a 0 1 1 1
b 0 1 1 1
c 0 1 2 2
d 0 1 2 2
e 0 1 2 3

Solution 2 – Space Optimized DP

We can optimize space by using only two rows of the DP table.

public int longestCommonSubsequence(String text1, String text2) { int m = text1.length(); int n = text2.length(); int[] prev = new int[n + 1]; int[] curr = new int[n + 1]; for (int i = 1; i <= m; i++) { for (int j = 1; j <= n; j++) { if (text1.charAt(i - 1) == text2.charAt(j - 1)) { curr[j] = prev[j - 1] + 1; } else { curr[j] = Math.max(prev[j], curr[j - 1]); } } // Swap arrays int[] temp = prev; prev = curr; curr = temp; } return prev[n]; }
Time Complexity: O(m*n)
Space Complexity: O(n)

Solution 3 – Recursive with Memoization

Top-down approach using recursion and memoization:

public int longestCommonSubsequence(String text1, String text2) { int m = text1.length(); int n = text2.length(); int[][] memo = new int[m][n]; for (int i = 0; i < m; i++) { Arrays.fill(memo[i], -1); } return lcs(text1, text2, m - 1, n - 1, memo); } private int lcs(String text1, String text2, int i, int j, int[][] memo) { if (i < 0 || j < 0) { return 0; } if (memo[i][j] != -1) { return memo[i][j]; } if (text1.charAt(i) == text2.charAt(j)) { memo[i][j] = lcs(text1, text2, i - 1, j - 1, memo) + 1; } else { memo[i][j] = Math.max( lcs(text1, text2, i - 1, j, memo), lcs(text1, text2, i, j - 1, memo) ); } return memo[i][j]; }
Time Complexity: O(m*n)
Space Complexity: O(m*n) for memoization + O(m+n) for recursion stack

Printing the LCS

To print the actual LCS, we can backtrack through the DP table:

public String printLCS(String text1, String text2) { int m = text1.length(); int n = text2.length(); int[][] dp = new int[m + 1][n + 1]; // Fill DP table for (int i = 1; i <= m; i++) { for (int j = 1; j <= n; j++) { if (text1.charAt(i - 1) == text2.charAt(j - 1)) { dp[i][j] = dp[i - 1][j - 1] + 1; } else { dp[i][j] = Math.max(dp[i - 1][j], dp[i][j - 1]); } } } // Backtrack to find the LCS StringBuilder lcs = new StringBuilder(); int i = m, j = n; while (i > 0 && j > 0) { if (text1.charAt(i - 1) == text2.charAt(j - 1)) { lcs.insert(0, text1.charAt(i - 1)); i--; j--; } else if (dp[i - 1][j] > dp[i][j - 1]) { i--; } else { j--; } } return lcs.toString(); }
