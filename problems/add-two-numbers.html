Add Two Numbers

LeetCode Problem 2
Problem: You are given two non-empty linked lists representing two non-negative integers. The digits are stored in reverse order, and each of their nodes contains a single digit. Add the two numbers and return the sum as a linked list.

Note: The digits are stored in reverse order, meaning the 1's digit is at the head of the list.

Example:
Input: l1 = [2,4,3], l2 = [5,6,4]
Output: [7,0,8]
Explanation: 342 + 465 = 807

Solution 1 – Iterative Approach

Traverse both lists simultaneously, add corresponding digits and handle carry.

public ListNode addTwoNumbers(ListNode l1, ListNode l2) { ListNode dummy = new ListNode(0); ListNode current = dummy; int carry = 0; while (l1 != null || l2 != null || carry != 0) { int sum = carry; if (l1 != null) { sum += l1.val; l1 = l1.next; } if (l2 != null) { sum += l2.val; l2 = l2.next; } carry = sum / 10; current.next = new ListNode(sum % 10); current = current.next; } return dummy.next; }
Time Complexity: O(max(m, n)) where m, n are lengths of the lists
Space Complexity: O(max(m, n)) for the result list

Solution 2 – Recursive Approach

Use recursion to add the numbers, passing carry as a parameter.

public ListNode addTwoNumbers(ListNode l1, ListNode l2) { return addTwoNumbersHelper(l1, l2, 0); } private ListNode addTwoNumbersHelper(ListNode l1, ListNode l2, int carry) { if (l1 == null && l2 == null && carry == 0) { return null; } int sum = carry; if (l1 != null) { sum += l1.val; l1 = l1.next; } if (l2 != null) { sum += l2.val; l2 = l2.next; } ListNode result = new ListNode(sum % 10); result.next = addTwoNumbersHelper(l1, l2, sum / 10); return result; }
Time Complexity: O(max(m, n))
Space Complexity: O(max(m, n)) for recursion stack
Step-by-step Addition:

l1: 2 → 4 → 3 (represents 342)
l2: 5 → 6 → 4 (represents 465)

Step 1: 2 + 5 = 7, carry = 0
Result: 7

Step 2: 4 + 6 = 10, carry = 1
Result: 7 → 0

Step 3: 3 + 4 + 1 = 8, carry = 0
Result: 7 → 0 → 8

Final: 7 → 0 → 8 (represents 807)

Solution 3 – In-place Addition (If allowed to modify input)

Modify the longer list in-place to save space.

public ListNode addTwoNumbers(ListNode l1, ListNode l2) { int len1 = getLength(l1); int len2 = getLength(l2); // Make l1 the longer list if (len1 < len2) { ListNode temp = l1; l1 = l2; l2 = temp; } ListNode result = l1; ListNode prev = null; int carry = 0; while (l1 != null) { int sum = l1.val + carry; if (l2 != null) { sum += l2.val; l2 = l2.next; } l1.val = sum % 10; carry = sum / 10; prev = l1; l1 = l1.next; } if (carry > 0) { prev.next = new ListNode(carry); } return result; } private int getLength(ListNode head) { int length = 0; while (head != null) { length++; head = head.next; } return length; }
Time Complexity: O(max(m, n))
Space Complexity: O(1) excluding the result
More Examples:

Input: l1 = [0], l2 = [0]
Output: [0]

Input: l1 = [9,9,9,9,9,9,9], l2 = [9,9,9,9]
Output: [8,9,9,9,0,0,0,1]
Explanation: 9999999 + 9999 = 10009998

Solution 4 – Handle Different Lengths Efficiently

Optimized version that handles lists of different lengths more efficiently.

public ListNode addTwoNumbers(ListNode l1, ListNode l2) { ListNode dummy = new ListNode(0); ListNode current = dummy; int carry = 0; // Add digits while both lists have nodes while (l1 != null && l2 != null) { int sum = l1.val + l2.val + carry; carry = sum / 10; current.next = new ListNode(sum % 10); current = current.next; l1 = l1.next; l2 = l2.next; } // Handle remaining digits in l1 while (l1 != null) { int sum = l1.val + carry; carry = sum / 10; current.next = new ListNode(sum % 10); current = current.next; l1 = l1.next; } // Handle remaining digits in l2 while (l2 != null) { int sum = l2.val + carry; carry = sum / 10; current.next = new ListNode(sum % 10); current = current.next; l2 = l2.next; } // Handle final carry if (carry > 0) { current.next = new ListNode(carry); } return dummy.next; }
Time Complexity: O(max(m, n))
Space Complexity: O(max(m, n))

Edge Cases and Validation

Robust implementation handling edge cases:
