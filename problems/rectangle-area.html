Rectangle Area

LeetCode Problem N/A
Problem: Given the coordinates of two rectilinear rectangles in a 2D plane, return the total area covered by the two rectangles.

Solution 1 – Geometry

Calculate area with overlap consideration.

public int computeArea(int ax1, int ay1, int ax2, int ay2, int bx1, int by1, int bx2, int by2) { int area1 = (ax2 - ax1) * (ay2 - ay1); int area2 = (bx2 - bx1) * (by2 - by1); int overlapWidth = Math.min(ax2, bx2) - Math.max(ax1, bx1); int overlapHeight = Math.min(ay2, by2) - Math.max(ay1, by1); int overlapArea = 0; if (overlapWidth > 0 && overlapHeight > 0) { overlapArea = overlapWidth * overlapHeight; } return area1 + area2 - overlapArea; }
Time Complexity: O(1)
Space Complexity: O(1)
